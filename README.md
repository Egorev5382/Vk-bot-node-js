# VK Chat Bot (Node.js + vk-io)

Модульный бот для модерации бесед ВКонтакте: баны, муты, ники в чате, удаление сообщений, закрепление и многое другое.

Работает в 2026 году на vk-io 4.10.1.

## Возможности

- **Модерация**:
  - /ban @user [причина] — бан (удаление из беседы + запись в БД)
  - /unban @user — разбан + приглашение обратно
  - /mute @user минуты [причина] — мут (удаление сообщений до конца времени)
  - /unmute @user — размут
  - /kick @user — кик без записи в БД
  - /checkpunish [@user] — просмотр активных наказаний
  - /delmessage — удалить сообщение (по ответу)
  - /multidell <кол-во> — удалить последние N сообщений
  - /pin — закрепить сообщение (по ответу)

- **Ники в чате**:
  - /nick <ник> — установить свой ник
  - /setname @user <ник> — установить ник другому (админ)
  - /clearnick — очистить свой ник
  - /nicks — список всех ников
  - /nonicks — кто не поставил ник

- **Настройки беседы**:
  - /changename <название> — изменить название беседы
  - /set_vid <число> — установить VID (кастомное поле)
  - /chat — посмотреть текущие настройки чата

- **Прочее**:
  - /post <текст> — создать пост в группе (от имени группы)
  - /inchat @user — проверить, находится ли пользователь в беседе
  - /ping — проверка бота
  - /help — этот список
